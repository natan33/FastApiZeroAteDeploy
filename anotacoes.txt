     4. Instale o pyteste para poder fazer os teste na aplicação

           - pip install pytest
           - pip freeze -> para pegar a versão do pytest e colocar no arquivo requirements.txt

           - instale a versão pelo requirements.txt
                pip install -r requirements.txt
                - Caso dê algum erro 'ERROR: Could not open requirements file: [Errno 2] No such file or directory: 'requirements.txt''
                    digite ->> pip freeze > requirements.txt
                    depois digite ->> pip instal -r requirements.txt

     5. Agora vamos inicar o pytest no terminal

        - No terminal digite -> pytest
        - Veremos que deu um erro, pôs não está identado igual como o FastAPI Deixa o programa
        - Para isso vamos executar no terminal
            python -m pytest
         _ vamos saber a aplicaçãio esta funcionando corretamente ou não
            - caso não esteja veja qual o erro e tente concerta
